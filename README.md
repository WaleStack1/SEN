 Flutter Calculator — Assignment Submission

## Architecture
- **Pattern:** setState + Material Design 3
- **State management:** setState
- **Calculator type:** Scientific (Bonus)
- **Color theme:** golden_dark

## Features

- Addition, Subtraction, Multiplication, Division
- Percentage calculation
- Sign toggle (+/−)
- Backspace / clear

**Bonus scientific features:**
- Trigonometric: sin, cos, tan, asin, acos, atan (degrees mode)
- Hyperbolic: sinh, cosh, tanh
- Logarithms: log₁₀, ln
- Power: x², x³, 1/x, |x|, √x
- Constants: π, e
- Factorial: n!
- Combinatorics: nPr, nCr

flutter pub get
flutter run
